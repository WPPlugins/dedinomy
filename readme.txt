Avec le plugin Dédinomy, afficher toutes vos dédicaces sur votre installation Wordpress en quelques secondes.

== Description ==

Vous souhaitez afficher vos dédicaces via Dédinomy sur votre installation Wordpress, le plugin
Dédinomy for Wordpress est fait pour vous :
Grâce à notre widget, vous pouvez facilement ajouter la liste des dernières dédicaces en provenance de Dédinomy.
Vous n'avez plus qu'a réadapté votre feuille de style pour personnaliser l'affichage des dédicaces.

Vous pouvez utiliser réutilisez l'API dédinomy en dehors du Plugin pour libérer à 100 % 
l'affichage des dédicaces.

== Installation ==

= Pré-requis pour l'utilisation du plugin =

Ce plugin n'est compatible que pour les versions supérieurs à 4.0.0 de Dédinomy.
Pour vérifier votre version, connectez-vous à votre installation Dédinomy, à la rubrique Mon compte Dédinomy. Le numéro de version vous sera indiqué.
Pensez à mettre à jour votre version Dédinomy ainsi que celle de Wordpress avant l'installation du Plugin

= Installation =

1. Ajouter `dedinomy` dans le dossier `/wp-content/plugins/`
1. Activer le plugin dans le menu Plugins de Wordpress
1. Activer le widget

== Frequently Asked Questions ==

= Rien ne fonctionne ! =

Simplifions la tâche, si rien ne fonctionne : 
Vérifiez tout d'abord que votre version de Dédinomy est à jour (Version 4.0.0 et supérieurs);
Vérifiez la présence du fichier api.php sur votre dossier d'installation
Vérifiez en désactivant le plugin si l'erreur persiste. Dans ce cas, celle ci provient d'un autre plugin
